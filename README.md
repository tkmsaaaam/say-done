# say-done
say-done is a notification tool. It says "Done" within 10 seconds after another command is finished.

## Installation
```sh
git clone git@github.com:tkmsaaaam/say-done.git
cd say-done
cargo build
```

## How to use
```sh
./target/debug/say-done -c <COMMAND>
```
